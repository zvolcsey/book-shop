## Routing

- Home page (URL: /)
- Books list page (URL: /books)
- Detailed book page (URL: /book-name-slug + book-unique-id)
- Author page (URL: /author-name-slug + author-unique-id)
- Cart page (URL: /cart)

## Run locally

You need have Node v18.13.0 or higher.

```sh
cd client
npm ci
npm run dev
```

## Port

Client uses port 5173.
